Vocabulary of Cryptocurrency:

Blockchain: Distributed ledgers that are secured by cryptography. They act as public databases that can be read by anyone, but only

updated by data owners. The data is copied across thousands of computers worldwide instead of residing in a centralized server.

Node: A computer that possesses and maintains a copy of the blockchain.

Mining: The process of solving the next block, requiring large amounts of processing power but is rewarded with cryptocurrency.

Proof-of-work: The current consensus algorithm used by a cryptocurrency.

Software wallet storage of cryptocurrency that exists as software files on a computer

Hardware wallet: A device used to store cryptocurrency securely, often regarded as the most secure storage of cryptocurrencies.

More detailed explanations of these concepts and more can be found at Cryptominded.com
